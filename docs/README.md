**Project:** *OpenLingua — Language School Website (Exam Work)*

- Single‑page responsive website for a language school with a separate student account page.
- Built with **HTML5**, **CSS3**, **Bootstrap 5**, **JavaScript**, and an interactive map via **Leaflet + OpenStreetMap**.
- Main page includes: hero section with CTA, “About” block, benefits cards, searchable course catalog with levels, tutor list with filters, interactive map, and contacts.
- Student account page shows a paginated table of orders with modals for **viewing details**, **editing**, and **deleting** bookings.
- Implemented as an **exam project** to demonstrate layout skills, responsive design, UI components, client‑side filtering, and basic CRUD operations for course/tutor bookings.
